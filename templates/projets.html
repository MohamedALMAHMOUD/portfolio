Projet 1: Flask API

J'essayais d'éxecuter un code python, ici, dans les pages GitHub. Le code de python tente de vérifier si l'entrée d'un "input" respecte bien certaines conditions:
- avoir au moins une Majuscule;
- avoir au moins 5 chiffres;
- avoir moins de 15 caractères
Mais malheureusement, ces pages sont statistiques et n'acceptent pas le script python. Donc, il faut un serveur python pour éxecuter le code. La solution est d'utiliser un mini framework flask. Aussi, on a besoin de déployer flask en ligne (production). Je cherchais une solution gratuite et une alternative à Herkou qui devient pyant. Heureusement, Render est là pour déployer mon projet hébergé sur GitHub. On peut trouver le projet complet ICI.
Pour tester directement le projet c'est simple, on peut aller sur le site, on tape un mot de passe et on appuie sur le bouton. Ensuite, on aura le résultat si le mot de passe respecte ou non les conditions requises.
Aussi, je voluais integrer le projet, ici, dans mon portfolio. C'est-à-dire, je veux utiliser une API pour communiquer un front end (mon portfolio) et un back end , le serveur python (Render).
Enfin, en tapant un mot de passe dans le cadre ci-dessous puis cliquer sur le bouton, une requette HTTP envoie le mot de passe au serveur deployé sur Render via l'API. On reçoit la réponse et l'affiche juste en dessous du cadre.
⚠ : Juste une petite remarque. Comme le serveur est gratuit, il prend un peu de temps pour répondre.
